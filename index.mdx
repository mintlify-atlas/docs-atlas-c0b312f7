Defaults is a Swift library that makes `UserDefaults` pleasant to work with. You declare strongly-typed keys with default values in one place, and access them with a clean subscript syntax throughout your app. It's used in production by millions of users across [Sindre Sorhus's apps](https://sindresorhus.com/apps).

## Why Defaults?

Defaults gives you a type-safe façade over `UserDefaults` with a rich set of built-in features:

<CardGroup cols={2}>
  <Card title="Strongly typed" icon="shield-check">
    Declare the type and default value once. The compiler enforces correctness everywhere you use the key.
  </Card>
  <Card title="SwiftUI integration" icon="apple">
    Use `@Default` like `@State` — views automatically re-render when values change.
  </Card>
  <Card title="Codable & NSSecureCoding" icon="database">
    Store enums, structs, and any `Codable` or `NSSecureCoding` type out of the box.
  </Card>
  <Card title="iCloud sync" icon="cloud">
    One flag on a key and it automatically syncs across all your user's devices.
  </Card>
  <Card title="Async observation" icon="bolt">
    Observe changes with `async/await` using `Defaults.updates()` returning an `AsyncStream`.
  </Card>
  <Card title="Custom types" icon="puzzle-piece">
    Implement `Defaults.Bridge` to serialize any custom type exactly the way you want.
  </Card>
</CardGroup>

## Getting started

<Steps>
  <Step title="Add the package">
    In Xcode, go to **File → Add Package Dependencies** and enter `https://github.com/sindresorhus/Defaults`.
  </Step>
  <Step title="Define your keys">
    Extend `Defaults.Keys` to declare strongly-typed keys with default values.

    ```swift
    import Defaults

    extension Defaults.Keys {
        static let quality = Key<Double>("quality", default: 0.8)
        static let isUnicornMode = Key<Bool>("isUnicornMode", default: false)
    }
    ```
  </Step>
  <Step title="Read and write values">
    Access stored values anywhere in your app using the `Defaults` subscript.

    ```swift
    Defaults[.quality]       // => 0.8
    Defaults[.quality] = 0.5 // => 0.5
    ```
  </Step>
  <Step title="Use in SwiftUI">
    Bind keys directly to your views with the `@Default` property wrapper.

    ```swift
    struct ContentView: View {
        @Default(.quality) var quality

        var body: some View {
            Slider(value: $quality, in: 0...1)
        }
    }
    ```
  </Step>
</Steps>
